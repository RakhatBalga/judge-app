# Языковое Меню Профиля - Реализовано ✅

## Описание

Добавлено красивое профильное меню с функциями переключения языка и выхода из аккаунта. Меню появляется при нажатии на аватар судьи в верхнем правом углу страницы команд.

## Основные Характеристики

### 1. Язык по Умолчанию - Казахский 🇰🇿
- При первом входе на сайт отображается казахский язык (kk)
- Выбор языка сохраняется в localStorage
- При повторном входе восстанавливается сохраненный язык

### 2. Профильное Меню
```
Компонент: ProfileMenu.vue
Локация: src/shared/ui/ProfileMenu.vue
Использование: TeamsPage.vue (в header)
```

Меню содержит:
- **Профиль судьи** - Кликабельная кнопка с аватаром и именем
- **Выпадающее меню** - Открывается при клике с анимацией
- **Выбор языка** - Три языка с флагами стран
- **Кнопка выхода** - Красная кнопка выхода из аккаунта

### 3. Доступные Языки

| Флаг | Название    | Код | Статус      |
|------|-------------|-----|-------------|
| 🇬🇧  | English     | en  | Доступен    |
| 🇷🇺  | Русский     | ru  | Доступен    |
| 🇰🇿  | Қазақша     | kk  | По умолчанию|

### 4. Функционал Меню

**При открытии:**
- Плавная анимация появления (слайд вверх + fade in)
- Полупрозрачный фон закрывает остальной контент
- Меню позиционируется под кнопкой профиля

**При выборе языка:**
- Визуальный индикатор (✓) показывает текущий язык
- Текст языка выделен зеленым при выборе
- Все тексты на странице мгновенно обновляются
- Меню автоматически закрывается

**При выходе:**
- Пользователь выходит из аккаунта
- Перенаправляется на страницу логина
- Меню закрывается

**При клике вне меню:**
- Меню закрывается автоматически
- Фон исчезает

### API Компонента

Компонент полностью автономный и не требует props. Внутри использует:

```typescript
// Из useI18n()
- t()              // Функция перевода
- locale           // Текущий язык (реактивная переменная)
- setLocale()      // Функция смены языка
- availableLocales // Массив доступных языков

// Из auth store
- auth.user        // Информация о пользователе
- auth.logout()    // Функция выхода

// Из router
- router.push()    // Навигация
```

## Дизайн и Стили

### Цветовая Схема:
- **Основной цвет (Mint)**: #28ca9e / кк для активного состояния
- **Фон меню**: Белый (bg-white)
- **Текст**: Грифель (slate-600 и выше)
- **Кнопка выхода**: Красная (red-600)
- **Селектор языка**: Зеленый при выборе

### Анимации:

## Поведение

### Сценарий 1: Первый вход на сайт
1. Пользователь логинится
2. На странице Teams видит казахский язык (Қазақша)
3. Весь интерфейс отображается на казахском

### Сценарий 2: Смена языка
1. Пользователь кликает на аватар
2. Открывается меню с тремя языками
3. Выбирает нужный язык (например, Русский)
4. Весь интерфейс мгновенно переводится на русский
5. Язык сохраняется в localStorage

### Сценарий 3: Выход из аккаунта
1. Пользователь кликает на аватар
2. Нажимает красную кнопку "Выход"
3. Выходит из аккаунта
4. Перенаправляется на страницу логина

### Сценарий 4: Закрытие меню
1. Пользователь кликает вне меню
2. Меню закрывается с анимацией
3. Фоновый фильтр исчезает

## Технические Детали

### React 18 (Composition API):
- `ref()` для состояния меню
- `useRouter()` для навигации
- `useAuthStore()` для управления аутентификацией

### Transitions:
- `<Transition name="menu">` - Анимация меню
- `<Transition name="fade">` - Анимация фона

### Состояния:
- `isMenuOpen` - Открыто ли меню
